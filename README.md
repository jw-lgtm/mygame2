# 旁白永远是对的

一个纯前端单页网页小游戏：流程图迷宫 + 毒舌旁白 + 多结局互动叙事。

## 文件结构

```text
.
├── index.html
├── css/
│   └── styles.css
└── js/
    ├── audio.js
    ├── game.js
    ├── narrator.js
    ├── rooms.js
    ├── save.js
    └── ui.js
```

## 运行

直接双击 `index.html` 可以本地预览。

如果要用 https 链接发给微信好友，把整个文件夹部署到 Netlify、Vercel、GitHub Pages 或任意静态服务器即可。入口文件是 `index.html`。

## 已实现

- 34 个流程图房间节点
- 8 个结局：6 个常规结局 + 2 个隐藏结局
- 100+ 段中文旁白
- 动态旁白状态：好感度、愤怒值、控制欲
- 连续三次违抗旁白后解锁隐藏反抗路线
- 连续五次听从旁白后解锁隐藏听从路线
- 耐心值归零进入理智崩溃结局
- localStorage 自动存档
- 成就与统计
- Web Audio API 动态配乐和音效
- 假错误、假崩溃、文字闪烁、界面抖动、按钮偏移等沉浸破坏演出
- 手机和桌面自适应
